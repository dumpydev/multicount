# MultiCount!
Simple and easy to use counter for various purposes.
## Features
 - [x] Open Source
 - [x] Easy to use
 - [x] Made in flutter with material design
## Installation
```bash
git clone https://github.com/dumpydev/multicount.git
cd multicount
flutter pub get
flutter build apk
adb install build/app/outputs/apk/app-release.apk
```
